Summary

mod_jsonindex The basic idea of this module is to provide easy access to the list of files located in a single directory.

The current HTML implementation with mod_autoindex is nice, but it lacks one major point, its output has to be additionally parsed.

With this module the list of files is directly JSON encoded and you can very simply implement this into your AJAX web applications.

Enabling JSONindex Support

To enable the jsonindex only for browsers from the example.com domain add this code to your httpd.conf configuration file

<Location /somedir>
   SetHandler json-directory

Order Deny,Allow
Deny from all
Allow from .example.com
</Location>

<Directory /path/to/some/dir>
   SetHandler json-directory
</Directory>

JSON Pretty output

A 'pretty' encoded JSON version of the output is available by accessing the page:

   http://your.server.name/some_dir/?pretty

This is useful when debuging the JSON output.
